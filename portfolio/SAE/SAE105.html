SAE 1.05 — Traiter et visualiser des données ouvertes

Dans cette SAE, j’ai développé un script permettant de récupérer des données au format CSV depuis une plateforme open data, de les traiter puis de les convertir au format JSON afin de les exploiter dans une interface web de visualisation graphique.

Compétences sélectionnées

AC23.01
AC23.05

Pourquoi ces Compétences ?

J’ai associé cette SAE à AC23.01 car le cœur du projet reposait sur l’écriture d’un script automatisant la récupération, le traitement et la transformation de données.

J’ai également choisi AC23.05 car cette SAE nécessitait d’accéder à des ensembles de données externes et de les rendre exploitables pour une application web de visualisation.

Conclusion

Cette SAE m’a permis de renforcer mes compétences en scripting, en traitement de données et en manipulation de formats comme CSV et JSON. Elle m’a également aidé à mieux comprendre le cycle complet d’exploitation de données, depuis leur récupération jusqu’à leur visualisation graphique.
